# Tarea de Programación: Cálculo de Descuento en Compras

Este repositorio contiene la solución a la tarea de programación sobre cálculo de descuentos en compras.

## Descripción

El programa implementa una función llamada `calcular_descuento` que recibe:
- El monto total de la compra.
- Un porcentaje de descuento (por defecto es 10%).

La función devuelve el monto del descuento calculado.  
En el programa principal se realizan dos ejemplos de uso:  
1. Una compra con el porcentaje de descuento por defecto (10%).  
2. Una compra con un porcentaje específico (20%).  

## Archivos

- `descuento.py`: Código fuente del programa en Python.  
- `tarea_descuento.zip`: Archivo comprimido con el código (listo para subir a Moodle).
